Currency Property

Description

The Currency dataproperty consists of a dropdown showing the currencies available. The property can be used standalone or as bound to a dataobject.

The available currencies are given by the items of the currencies dataobject, which is created when the currency property is installed. The dataobject stores its items in dynamic storage. Items can be added, changed or deleted via the dynamicdata admin interface.

The currencies dataobject has the following properties: ID, iso_code, numeric_code, name, country, symbol, decimalseparator, dsalwaysshown, groupingseparator, groupingsize, minfractiondigits, maxfractiondigits, minintegerdigits, maxintegerdigits, precision, rounding, zerodigitsign, and minussign.

The currency dataproperty presently does not have its own templates, but uses those of the parent dropdown property.

Usage

Standalone

    <xar:data-input type="currency" [attr1="value1",...] />
    <xar:data-output type="currency" [attr1="value1",...] />

Bound

    <xar:data-input property="$properties['currency']" [attr1="value1",...] />
    <xar:data-output property="$properties['currency']" [attr1="value1",...] />

Examples

    <xar:data-input type="currency" display_prop="iso_code" />
    <xar:data-input type="currency" store_prop="name" />

Attributes

  • display_prop: string [default='name'] Defines the object property to be displayed in the dropdown.
  • store_prop: string [default='iso_code'] Defines the object property to be used in the dropdown values.
