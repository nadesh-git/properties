IBAN Property

Description

The ESR dataproperty is a textbox property ESR number code. It lets you verify that the code entered is a correct ESR number.

Usage

Standalone

    <xar:data-input type="esr" />
    <xar:data-output type="esr" />

Bound

    <xar:data-input property="$properties['esr']" />
    <xar:data-output property="$properties['esr']" />

Attributes

  • None
