CelkoPosition Property

Description

The CelkoPosition dataproperty is a widget that displays the parameters of a celko position for an item.

This property is useful when your items are arranged in so called nested sets. For more information about nested sets see here. The property displays two dropdowns in an item's input form:

  • A list of all items in the tree
  • A list of possible relations to of this item to the item chosen in the first dropdown

Given this information the item will be correctly inserted into a given position in the tree.

This property can only be used as bound to an object.

Configuration

  • Table: The database table the celko information is stored in.
  • Name: The field in that table that holds name information of its items.
  • Parent ID: The field in that table that holds the id of the parent of a given item.
  • Left ID: The field in that table that holds the id of the field "to the left" of a given item.
  • Right ID: The field in that table that holds the id of the field "to the right" of a given item.

Further configuration options supported are those inherited from the parent base property.

Version

This dataproperty is not version specific and works with any version of Xaraya 2.2.0 or later.
