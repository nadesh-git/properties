Address Property

Description

The Address dataproperty lets you manage mailing addresses. A mailing address has the folowing parts:

  • One or more lines for street, P.O. Box, apartment etc.
  • City
  • Province or state
  • Postal code
  • Country

Usage

Standalone

    <xar:data-input type="address" [attr1="value1",...] />
    <xar:data-output type="address" [attr1="value1",...] />

Bound

    <xar:data-input property="$properties['address']" [attr1="value1",...] />
    <xar:data-output property="$properties['address']" [attr1="value1",...] />

Attributes

  • rows: integer [default=2] The number of rows displayed for the street part of the address
  • show_city: boolean [default=true] Display the city part of the address
  • show_province: boolean [default=true] Display the state/province part of the address
  • show_postal_code: boolean [default=true] Display the postal code part of the address
  • show_country: boolean [default=true] Display the country part of the address

Methods

getValueArray(): Returns an associative array of the address components, with the keys line_{1..n}, city, province, postal_code, country.
