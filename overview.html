Time Property

Description

The Time dataproperty lets you input times. It consists of text and/or dropdown fields. The fields define hours, minutes and seconds.

The property stores times internally as seconds.

Usage

Standalone

    <xar:data-input type="time" [attr1="value1",...] />
    <xar:data-output type="time" [attr1="value1",...] />

Bound

    <xar:data-input property="$properties['time']" [attr1="value1",...] />
    <xar:data-output property="$properties['time']" [attr1="value1",...] />

Examples

    <xar:data-input type="time" layout="byhour" />
    <xar:data-input type="time" format="ampm" />

Attributes

  • layout: string [default='default'] The default and byminute layouts show 3 text fields for hours, minutes and seconds in 12 or 24 hour format. The byhalfhour layout shows a single dropdown displaying half hourly increments in 24h format. The byhour layout shows a single dropdown displaying hourly increments in 24h format.
  • format: string [default='24'] If the value is 'ampm' then a am/pm dropdown is displayed.
