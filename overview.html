Autocomplete Property

Description

The Autocomplete dataproperty is a text field that displays dropdown values from a database table. As the user types characters into the field, hints that match the characters are displayed as a dropdown.

This property extends the textbox property and is based on the Devbridge autocomplete jquery plugin. (see the code included in this property's folder.)

  • The property uses AJAX calls to query the database for hints. It does this by calling a module function using ws.php. The module function must be of the type [module_name]_wsapi_[function_name].
  • The returned data from the database is an associative array modelling a dropdown following the usual convention. The conventional store and display fields are the fields id and name. Alternate field names can be defined in the call as store and display attributes respectively.

Usage

Unbound

    <xar:data-output type="autocomplete" modurl="value1" modfunc="value1" [store="value3"] [display="value4"]/>

Attributes

  • urlmod: string  The name of the module of the call function
  • urlfunc: string  The name of the function of the call function
  • store: string  The name of database field holding the value of the dropdown items
  • display: string  The name of database field holding the display value of the dropdown items
