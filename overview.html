TimeFrame Property

Description

The TimeFrame dataproperty is a dropdown that displays past and/or future timeframes.

The property's values are integers representing references to timeframes, and not in any way timeframes themselves.

Usage

Standalone

    <xar:data-input type="timeframe" [attr1="value1",...] />
    <xar:data-output type="timeframe" [attr1="value1",...] />

Bound

    <xar:data-input property="$properties['timeframe']" [attr1="value1",...] />
    <xar:data-output property="$properties['timeframe']" [attr1="value1",...] />

Examples

    <xar:data-input type="timeframe" frames="future, past" />

Attributes

  • frames: mixed [default='past'] A list or array of sets of timeframes. 'past' and 'future' are included in the defualt templates. Unrecognized frames are ignored.
    • 'past' has the following values:
      DisplayValue
      This Month0
      Last Month-100
      Last 2 Months-200
      Last Quarter-300
      Year to Date-400
      Last Year-500
    • 'future' has the following values:
      DisplayValue
      Current Year500
      Till Yearend400
      This Quarter300
      Next 2 Months200
      Next Month100
  • Other attributes as inherited from the parent dropdown dataproperty.
