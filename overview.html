Secureemail Property

Description

The Secureemail dataproperty lets you obfuscate emails on web pages. It displays a mailto link which opens a mail client page with an email address. The email address is obfuscated in the web page on which it appears and so cannot be read by email harvesters.

The property uses a simple obfuscation method, splitting the email address into parts and then reassembling the address when the link is clicked.

Usage

Standalone

    <xar:data-input type="secureemail" [attr1="value1",...] />
    <xar:data-output type="secureemail" [attr1="value1",...] />

Bound

    <xar:data-input property="$properties['secureemail']" [attr1="value1",...] />
    <xar:data-output property="$properties['secureemail']" [attr1="value1",...] />

Attributes

  • value: string [default=''] The email address to be obfuscated. Clicking on an email link will open a page in your mail client with this address as the recipient.
  • label: string [default=''] An optional display name for the email link. If not set, the link will display the email address.
