JQDateTime Property

Description

The JQDateTime dataproperty is similar to the DateTime property, except that it uses a JQuery plugin for input. See the DateTime property for details.

Usage

Standalone

    <xar:data-input type="jqdatetime" [attr1="value1",...] />
    <xar:data-output type="jqdatetime" [attr1="value1",...] />

Bound

    <xar:data-input property="$properties['jqdatetime']" [attr1="value1",...] />
    <xar:data-output property="$properties['jqdatetime']" [attr1="value1",...] />
