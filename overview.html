DateTime Property

Description

The DateTime dataproperty lets you input times displayed as dates and times. It consists of a series of dropdown and a text fields.

The property stores times internally as seconds.

Usage

Standalone

    <xar:data-input type="datetime" [attr1="value1",...] />
    <xar:data-output type="datetime" [attr1="value1",...] />

Bound

    <xar:data-input property="$properties['datetime']" [attr1="value1",...] />
    <xar:data-output property="$properties['datetime']" [attr1="value1",...] />

Attributes

  • rows: integer [default=2] The number of rows displayed for the street part of the address
  • show_city: boolean [default=true] Display the city part of the address
  • show_province: boolean [default=true] Display the state/province part of the address
  • show_postal_code: boolean [default=true] Display the postal code part of the address
  • show_country: boolean [default=true] Display the country part of the address
