Number Property

Description

The Number dataproperty manages formatted numbers. It lets you enter and display numbers in a given format.

Usage

Standalone

    <xar:data-input type="number" [attr1="value1",...] />
    <xar:data-output type="number" [attr1="value1",...] />

Bound

    <xar:data-input property="$properties['number']" [attr1="value1",...] />
    <xar:data-output property="$properties['number']" [attr1="value1",...] />
