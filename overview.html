BIC Property

Description

The BIC dataproperty is a textbox property for BIC codes. It lets you verify that the code entered is a correct BIC code.

Usage

Standalone

    <xar:data-input type="bic" />
    <xar:data-output type="bic" />

Bound

    <xar:data-input property="$properties['bic']" />
    <xar:data-output property="$properties['bic']" />

Attributes

  • None

Attributes

The property is basically an extension of textbox with a validation for BIC codes.
