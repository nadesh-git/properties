CodeMirror Property

Description

The CodeMirror dataproperty is a standalone widget that lets you embed the CodeMirror code editor into your application. It doesn't have any configuration options at present.

The property is based on version 2 of CodeMirror

Usage

Standalone

    <xar:data-input type="codemirror" [attr1="value1",...] />

Attributes

The only attributes supported are those inherited from the parent textarea property.

If you want to configure the editor you need to tweak the javascript in the showinput template (line 11 and following). There are many examples on the CodeMirror site that show how this can be done..
