Country Property

Description

The Country dataproperty displays a dropdown of country names, optionally with the selected country's flag..

Usage

Standalone

    <xar:data-input type="country" [attr1="value1",...] />
    <xar:data-output type="country" [attr1="value1",...] />

Bound

    <xar:data-input property="$properties['country']" [attr1="value1",...] />
    <xar:data-output property="$properties['country']" [attr1="value1",...] />

Configuration Values

  • layout: string [default="default}"] A string defining the display layout for the property. Valid strings are "text", "image", "imagetext".
