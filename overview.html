LinkTrail Property

Description

The LinkTrail dataproperty displays a series of links of recently visited pages. It saves a list of the last 20 pages visited and displays the last 5 of these by default.

The number of pages to be displayed can be configured.

This property is meant to be used standalone. The input and output displays are the same.

Usage

Standalone

    <xar:data-input type="linktrail" [attr1="value1",...] />
    <xar:data-output type="linktrail" [attr1="value1",...] />

Examples

    <xar:data-output type="linktrail" links_to_show="3" />

Attributes

  • links_to_show: number [default=5] The number of recently visited pages to display.
