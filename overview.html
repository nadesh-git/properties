Flot Property

Description

The Flot dataproperty is a simple wrapper for the Flot JQuery graphic library

It is a standalone property, to be called through a property tag. It has no showinput method. Display is purely through the showoutput version of the tag.

Usage

Standalone

    <xar:data-output type="flot" [attr1="value1",...] />
