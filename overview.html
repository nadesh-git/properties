Province Property

Description

The Province dataproperty is a dropdown widget that displays a list of provinces of a specific country.

Usage

Standalone

    <xar:data-input type="province" country="$country" [attr1="value1",...] />

Bound

    <xar:data-input property="$properties['province']"  country="$country" [attr1="value1",...] />

Example

    <xar:data-input type="province" name="provinces" country="ca"/>

Attributes

  • country: char(2) [default='us'] The ISO code of the country of the provinces to be displayed. You can have the provinces of more than one country shown by inputting a comma separated list of ISO codes: 
        <xar:data-input type="province" country="us,ca"/>

Further attributes supported are those inherited from the parent dropdown property.
