CurrentTime Property

Description

The CurrentTime dataproperty displays the current time in a given timezone. The timezone can be changed via the property's UI.

Usage

Standalone

    <xar:data-input type="currenttime" [attr1="value1",...] />
    <xar:data-output type="currenttime" [attr1="value1",...] />

Bound

    <xar:data-input property="$properties['currenttime']" [attr1="value1",...] />
    <xar:data-output property="$properties['currenttime']" [attr1="value1",...] />

Configuration Values

  • Format: string lis [default="'Y-m-d H:i:s'"] A string representing the display format
