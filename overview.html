IBAN Property

Description

The IBAN dataproperty is a textbox property IBAN code. It lets you verify that the code entered is a correct IBAN.

Usage

Standalone

    <xar:data-input type="iban" />
    <xar:data-output type="iban" />

Bound

    <xar:data-input property="$properties['iban']" />
    <xar:data-output property="$properties['iban']" />

Attributes

  • None

Attributes

The property is basically an extension of textbox with a wrapper of the php-iban library
