Timeperiod Property

Description

The Timeperiod dataproperty lets you input time periods. It consists of a dropdown and a text field. The dropdown is meant for choosing the unit of time (seconds, minutes, hours, days, weeks, months and years). The textbox is meant for entering the number of units.

The property stores time periods as seconds. When displaying a period, it will choose the largest possible unit the period can be displayed in without decimals.

Usage

Standalone

    <xar:data-input type="timeperiod" [attr1="value1",...] />
    <xar:data-output type="timeperiod" [attr1="value1",...] />

Bound

    <xar:data-input property="$properties['timeperiod']" [attr1="value1",...] />
    <xar:data-output property="$properties['timeperiod']" [attr1="value1",...] />

Attributes

  • The attributes are those inherited from the parent base dataproperty.
