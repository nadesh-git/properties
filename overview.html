ObjectItem Property

Description

The ObjectItem dataproperty lets you choose a DD object and item from a series of dropdowns.

The object can be chosen from a dropdown that lists all the DD objects on the site. Initially the object is an empty string and the item value is 0, with no dropdown for it visible. Once an object is chosen, it's items become visible in a second dropdown

The object is stored by its name and displayed by its label. The item is stored by its ID and displayed by its name. If no name property is defined it is displayed as an ID.

The value of this property is stored and displayed as a string of the form [objectname]:[itemid].

This property has no configuration properties.

Usage

Standalone

    <xar:data-input type="objectitem" [attr1="value1",...] />
    <xar:data-output type="objectitem" [attr1="value1",...] />

Bound

    <xar:data-input property="$properties['objectitem']" [attr1="value1",...] />
    <xar:data-output property="$properties['objectitem']" [attr1="value1",...] />

Attributes

  • object: string [default=''] The name of the dataobject
  • item: integer [default=0] The ID of the item
  • value: string [default=':0'] A value of the form [objectname]:[itemid]
