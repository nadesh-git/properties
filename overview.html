Weeklist Property

Description

The Weeklist dataproperty displays a checkbox list of weekdays. The days are indicated (for now) by their starting letter (M, T, W, ...)

Usage

Standalone

    <xar:data-input type="weeklist" />
    <xar:data-output type="weeklist" />

Bound

    <xar:data-input property="$properties['weeklist']" />
    <xar:data-output property="$properties['weeklist']" />

Configuration Values

  • Monday first: integer [default=1] This defines which day of the week the checkbox list starts with. When this is set to 0, the first day in the week is Sunday.
